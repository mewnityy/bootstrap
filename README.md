# 🐾 Big Floppa Bootstrap

This project is a responsive single-page website created using Bootstrap 5, dedicated to the legendary meme character Big Floppa (Gregory the caracal). It includes information about Floppa, a gallery, achievements, an interactive profile editing form, and a photo slider.

---

## 📁 Project Structure

```
/
├── index.html            # Main profile page
├── storinka.html         # Achievements page
├── img/                  # Image folder (shlepa1.jpg, kartka.png, etc.)
├── logo.png              # Logo icon
└── README.md             # Project documentation
```

---

## 🧩 Technologies Used

- **HTML5** — page structure
- **Bootstrap 5.3** — grid, styles, components, responsive adaptation
- **Bootstrap Icons** — icons for menu, buttons, actions
- **JavaScript** — form validation, toast messages
- **Modal API** — for profile editing
- **Carousel API** — photo slider

---

## ⚙️ Main Sections

### 🔹 `navbar`
- Responsive navigation with logo, menu (Home, About, Achievements), and login button.
- Sticky top + mobile collapsible menu.

### 🔹 Floppa Profile
- Left — photo with border.
- Right — table with name, age, email, country.
- Edit/delete buttons.
- Modal with Bootstrap form that includes:
  - Name
  - Age
  - Email
  - Country
  - “About Me” description
- Toast notification after saving.

### 🔹 About Me
- Text block describing Big Floppa’s character.

### 🔹 Photo Carousel
- Slider with Floppa images.
- Built-in `16x9` aspect ratio.

### 🔹 `storinka.html` — Achievements
- Image cards (Bootstrap cards) in a grid.
- Each contains a photo, achievement title, date, and description.

---

## 💡 Features

- 🔄 Responsive (mobile + desktop)
- 🎯 Form field validation
- 🎞️ Carousel slider with swipe support
- 🧾 Profile data table
- 📌 Sticky navbar
- 🧁 Toast save confirmation

---

## 📦 How to Launch

> Open `index.html` in any modern browser. All dependencies (Bootstrap CSS, JS, and icons) are connected via CDN.

#  Big Floppa Bootstrap

Цей проєкт — адаптивна односторінкова вебсторінка, створена на Bootstrap 5, присвячена легендарному мемному персонажу Big Floppa (Григорій каракал). Вона містить інформацію про Floppa, галерею, досягнення, інтерактивну форму редагування профілю та слайдер з фото.

---

## 📁 Структура проєкту

```
/
├── index.html            # Основна сторінка профілю
├── storinka.html         # Сторінка з досягненнями
├── img/                  # Каталог зображень (shlepa1.jpg, kartka.png тощо)
├── logo.png              # Іконка логотипу
└── README.md             # Документація проєкту
```

---

## 🧩 Використані технології

- **HTML5** — структура сторінки
- **Bootstrap 5.3** — сітка, стилі, компоненти, responsive-адаптація
- **Bootstrap Icons** — піктограми меню, кнопок, дій
- **JavaScript** — валідація форм, тост-повідомлення
- **Modal API** — для редагування профілю
- **Carousel API** — слайдер з фотографіями

---

## ⚙️ Основні секції

### 🔹 `navbar`
- Адаптивна навігація з логотипом, меню (Дім, Про все, Досягнення) та кнопкою входу.
- Sticky top + мобільне згортання.

### 🔹 Профіль Floppa
- Ліворуч — фото з обводкою.
- Праворуч — таблиця з іменем, віком, email, країною.
- Кнопки редагування/видалення.
- Модальне вікно з Bootstrap-формою, що включає:
  - Ім’я
  - Вік
  - Email
  - Країна
  - Опис "Про себе"
- Toast-сповіщення після збереження.

### 🔹 Про мене
- Текстовий блок з описом персонажа Big Floppa.

### 🔹 Карусель фото
- Слайдер із зображеннями Floppa.
- Вбудоване співвідношення сторін `16x9`.

### 🔹 `storinka.html` — Досягнення
- Картки зображень (картки Bootstrap) у сітці.
- Кожна містить фото, назву досягнення, дату та опис.
